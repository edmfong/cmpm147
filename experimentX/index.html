Experiment 5 - Evolutionary Impressions

Description

This experiemnt generates images and mutates itself to optimize the best image it can generate.

Technical

The image takes the original image and draws a bunch of shapes. Depending on where the image was places, the generator reads the color of the position where the shape is placed, then it assigns that color to the shape. The generator takes three shapes and includes an image of each planet in our solar system. The user has the ability to upload their own images and change the mutation rate.
